# InnerScript User Review

Purpose: durable notes from Joel's hands-on review of the running app.

## 2026-06-03

- Title `Enter` should move cursor into the body.
- New Note should feel immediate and clearly create/select a real note.
- Empty title should not derive the sidebar title from body text.
- Sidebar note card should show only the note name; creation date should stay metadata.
- Body prompt should warmly invite writing with "How was your day?"
- Sidebar should be tighter and closer to an Obsidian file-tree style.
- Word count and character count in the editor footer are useful.
- Loading state should feel sleeker and less generic.
- Sidebar notes need clearer hover/click feedback.
- Interaction transitions should be smooth.
- Autosave status was flashing too fast, then became too hidden.
- Typed words sometimes disappear; this is a critical writing-trust bug.
- Header/title should not show a leading hashtag marker.
- User-review notes from app sessions should be documented in `.wolf` and loaded across sessions.
- Save status should react again on every further edit, not only the first save cycle.
